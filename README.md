# AI_Project - Sistema de Recomendación

## Descripción
Este proyecto consiste en la creación de un sistema de recomendación simple utilizando Python y GitHub Copilot.

## Objetivo
Aplicar herramientas de inteligencia artificial para generar recomendaciones.

## Herramientas utilizadas
- GitHub
- GitHub Copilot
- Python

## Proceso realizado
1. Creación de cuenta en GitHub
2. Activación de Copilot
3. Creación del repositorio
4. Desarrollo del código
5. Subida de archivos

## Funcionamiento
El sistema recomienda productos según preferencias del usuario.
